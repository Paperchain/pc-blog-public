    Ethereum Core Developers Have Reached Consensus on Hard Fork

    Posted on 2016, Oct 31 by Daniel Dewar One min read

    Last Friday, core developers for the Ethereum met to discuss possible courses of action to counter recent DDOS attacks on the network that were exploiting the platforms code logic.

    Consensus was reached for the following Ethereum Improvement Proposals (EIPs) to be included in an imminent hard fork:

    • EIP 155 - Simple replay attack protection
    • EIP 160 - EXP cost increase
    • EIP 161 - State trie clearing (invariant-preserving alternative)

    Users on the Mist platform will need to check frequently over the next couple of days for an update to the Ethereum client. Third party Ethereum clients will be reliable for pushing updates to the platform.
